Alzheimer Detection System using MRI and Clinical Data

This project is a Dual-Mode AI Framework for Alzheimer’s Disease Detection and Risk Assessment in rural healthcare.

Features

* MRI Mode → Predict Alzheimer Stage
* Clinical Mode → Predict Alzheimer Risk
* Dual Mode → Combined Prediction
* Streamlit Frontend Interface

Technologies Used

* Python
* TensorFlow / CNN
* Machine Learning
* Streamlit
* Scikit-learn
